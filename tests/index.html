Do screen readers switch pronunciation when language of a document is switched?

Warning: these results are all outdated, this project is archived.

This document contains a button labelled “Switch language”. If the button is activated the language of the page will be changed. Screen readers should change there pronunciation after the change to properly pronounce words. Test results are in the table on this page.

Test one: lang attribute on html element

If all content is switched out to the same language; changing the lang attribute on html element should suffice.

Example

This test is located in test-1.html. Expected result: screen reader should switch language/pronunciation. Before language change:

<html lang="en">...</html>

After language change:

<html lang="es">...</html>
